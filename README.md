# FalcomSchema
Schema repository for Kiseki/Ys games table files. Currently only supports the PC releases of the games on the latest engine. (Daybreak, Ys X)


# Game Naming Convention
The naming convention will be a combination of internal names and official english names. The full list is below:

- Trails through Daybreak: ed9_Daybreak1
- Trails through Daybreak 2: ed9_Daybreak2
- Ys X - Nordics: ys_X

Note for Kai no Kiseki: Kai will not officially be supported until an official english name is revealed. For all Kai related work please use the Kai branch.
